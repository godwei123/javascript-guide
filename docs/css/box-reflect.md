# box-reflect

-webkit-box-reflect CSS 属性可让你将元素内容在特定方向上进行轴对称反射。

语法：

```
box-reflect: <direction> <offset> <mask-box-image>;
```

- `<direction>`：指定反射的方向，可以是 `above`、`below`、`left` 或 `right`。
- `<offset>`：指定反射的偏移量，可以是长度值或百分比。
- `<mask-box-image>`：指定反射的遮罩图片。
