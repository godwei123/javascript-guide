# Lottie 动画

## 介绍

Lottie 是 Airbnb 开源的一个动画库，它可以将 Adobe After Effects 动画转换为 JSON 文件，然后在移动端播放。Lottie 支持 iOS、Android、Web 和 React Native。

::: code-group

```shell [安装]
npm install lottie-web
```

:::
